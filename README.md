# bou-ningen / 棒人間

ドットマトリクス LED 風の棒人間アニメーション。

## ページ

- `index.html` — **ドット棒人間**（メイン）。骨格を canvas のドットグリッドにラスタライズし、点灯/消灯ドットで描画。歩行アニメーション・RAINBOW・色/速度/ドット密度を調整可能。
- `line.html` — 旧ライン描画版（参照用）。`led-board-lite` から切り出した初代。

## ドット棒人間の仕組み

1. 棒人間を「頭(リング) + 胴/腕/脚(線分)」の骨格としてグリッド座標で定義 (`skeleton()`)。
2. 各ドットセルが骨格上にあるか距離判定 (`isLit()` / `distToSeg()`)。
3. 点灯ドットは発光、消灯ドットは薄く描いて LED パネルを再現。
4. 位相 `phase` で腕・脚を逆位相に振り、上下バウンド＋横移動で歩行。

## 由来

もとは `led-board-lite` の棒人間ボタン → `/bou-ningen` ページとして実装していたものを、
独立リポジトリに切り出し、ドット表現として作り直したもの。
